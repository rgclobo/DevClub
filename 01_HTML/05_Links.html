Links

A Tag <a> serve para criar Links.

Clique em mim por favor!

A tag <a>, acompanha o seu parametro, que é o 'href=""', que vai receber o caminho que vai ser enviado ao ser clicado.

Exemplo abaixo:

<a href="https://google.com">Vai para o Google.</a>

Vai para o Google

Link para página do mesmo site

É possivel usar a Tag <a> para criar um link de uma outra página do seu site.

<a href="./04_Tags-texto.html">Vamos para as Tag's de Texto</a>

Vamos para as Tag's de Texto

Link como âncora

É possivel usar a Tag <a>, para criar links como âncora no site, ao clicar, vai levar a parte que foi pré-definida utilizando o atributo ' id="" ', sem a necessidade de precisar ficar scrolando a página.

Muito útil para páginas com textos longos, assim é possivel navegar por pontos que são criados de maneira estratégica como âncoras. Exemplos abaixo:

<a href="#fim">Vamos para o fim da página.</a>

Vamos para o fim da página.

Parametro no link que abre em nova aba

Para você criar um link e ele abrir em uma nova aba, você vai escrever o link como mostrado acima, mas vai incluir o parametro target="_blank" após o final do link, ficando assim:

<a href="https://google.com" target="_blank">Vai para o Google.</a>

Vai para o Google em nova aba

Parametro no Link que envia um e-mail

Para você criar um link que o usuário vai clicar e enviar um e-mail para você, coloque o parametro mailto: antes do link, exemplo abaixo:

<a href="mailto:seuemail@seuemail.com">Envie um email</a>

Envie um email

Parametro no Link que faz ligação

Para criar um link que o usuário vai clicar e ligar para você, coloque o parametro tel: e o número correto que vai ser ligado, exemplo abaixo:

<a href="tel:+5541989876542">Me liga</a>

Me liga

Este é o fim da página, é aqui que vai ser colocado o parametro ' id="fim" '

O parametro ' id="" ' pode ser colocado em qualquer tag.

<p id="fim">Chegamos ao fim<p>

Chegamos ao fim

Também é possivel colocar o parametro ' id="" ' para ir para o inicio, ou qualquer lugar aonde ele for inserido, no exemplo abaixo, vamos voltar ao topo da página.

Perceba que agora o parametro ' id="" ' está na TAG <h1>

<h1 id="topo">Links</h1>

<a href="#topo">Vamos ao topo da página.</a>

Vamos ao topo da página.
