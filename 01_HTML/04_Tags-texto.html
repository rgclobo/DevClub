Principais TAG's para Texto

O Título mais importante do Site é o <h1> !

Os outros títulos não deixam de ser importantes! <h2>

Mas é importante sempre analisar para que os Motores de Busca (SEO), hanckeiem bem o site! <h3>

Além do h1 que é o mais importante, os títulos vão do h2 até o h6! <h4>

Com o tempo vai ficando mais claro qual e quando usar os outros tamanhos de títulos. <h5>
O importante é saber que existe uma lógica por trás disso. <h6>

Parágrafos <p>

Aqui são os paragrafos, para cria-los, é utilizado a tag <p> !

Aqui vai mais um parágrafo.

Aqui mais outro ...

Negrito <b>

Aqui tem um páragrafo com uma palavra em negrito.

A tag <b>, deixa o texto em negrito.

Itálico <i>

Aqui tem um parágrafo com uma palavra em itálico.

A tag <i>, deixa o texto em itálico.

Importante <strong>

Esse palavra é importante para o conteúdo.

Visualmente é como se fosse a tag <b>, mas a tag <strong>, mostra para os motores de busca (SEO), que é um termo ou uma frase importante no conteúdo.

Enfatizar texto <em>

Essa palavra está enfatizada neste parágrafo.

Visualmente é como se fosse a tag <i>, mas a tag <em>, serve para enfatizar um termo ou frase aos motores de busca. (SEO).
