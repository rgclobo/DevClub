Formulário

A tag <form>, é responsavel para agrupar os dados de um formulário e enviar ao mesmo tempo todos os <input> que estiverem dentro dele.

A propriedade action="" é responsavel por receber o endereço de onde vão ser enviados os dados do formulário.

a propriedade method="", tem dois atributos. A get e a post.

- method="get" é quando as informações do formulário são remetidas através da URL do site. É possivel velas ao dar enviar. Não é recomendavel e nem seguro quando se trata de dados de senha ou informações de cartão de crédito e afins.

- method="post" faz o envio das informações do formulário de forma encriptografadas. Maneira segura!

Para fazer o envio dos dados do arquivos, usamos a tag <button>. Nela temos algumas propriedades, as usadas abaixo são a type="reset" e a type="submit".

- type="reset" Vai limpar completamente todos os dados informados no seu formulário!

- type="submit" Vai fazer o envio dos dados do seu formulário!

Quando for vincular arquivos no seu formulário, pelo campo type="file" é necessário informar um atributo na tag <form>, o enctype="multipart/form-data, para que o servidor seja informado que vai receber arquivos neste formulário.

Exemplo:


        <form action="" method="get" enctype="multipart/form-data">
        <label for="name">Nome</label>
        <input type="text" name="name" id="name">

        <label for="email">Email:</label>
        <input type="email" name="email" id="email">

         <label for="file">Arquivo: </label>
         <input type="file" name="file" id="file">
        
        <button type="reset">Limpar</button>
        <button type="submit">Enviar</button>
        </form>
