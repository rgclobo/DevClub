Seletores

Um seletor Css é a forma que vamos conseguir linkar a estilização do CSS com o HTML.

Seletores pela Tag

Uma maneira de conseguir selecionar é através da tag utilizada.

No exemplo abaixo, iriamos estilizar a tag <h1> no arquivo .css.


        h1{
            color: blue;
            font-size: 20px;
        }

Lembre-se: Ao selecionar a tag que você vai estilizar, coloque a tag, abra e feche um colchetes ( {} ) e escreva as estilizações dentro. Assim como foi feito no exemplo acima.

Mas isso gera um problema, que ao fazer isso, toda e qualquer tag <h1>, irá receber a mesma estilização. No caso irá ficar na cor azul e com 20px de tamanho. E se quisermos selecionar mais especificamente?

Seletores por Classes (class="") e Identificador único (id="")

Classes (class="")

A classe é como se fosse um CEP do elemento. Serve para identificar um ou vários elementos HTML ao mesmo tempo. Ao criar classes pelo HTML, podemos selecionalas pelo CSS e estilizalas de uma maneira mais especifica.

Identificador único (id="")

O Identificador único (id=""), é como se fosse o CPF do elemento, pois só é possivel colocar um (id="") por elemento dentro de um HTML. Da mesma forma que a classe, ao criar ele no HTML, podemos selecionalo no CSS e estilizar. Mas sempre de maneira única!
