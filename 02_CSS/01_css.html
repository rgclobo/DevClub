Css

Para inserir os estilios de Css, pode ser através da nossa página do html, através da tag <style>, que deve ficar dentro da tag <head>.


        <style>
            h1{
                color: red;
                background-color: aqua;
            }
    
            h1 + p{
                color: blue;
                font-size: 20px;
            }
    
            button{
                background-color: green;
                border-color: darkgreen;
                border-radius: 5px;
                padding: .5em;
            }
    
            .upper{
                text-transform: uppercase;
            }
            </style>

É possivel também, criar um vínculo de um arquivo .css, externo. Também dentro da tag <head> é inserido o exemplo abaixo.


    <link rel="stylesheet" href="./01_css.css">

E dentro desse arquivo .css, que vai ficar todas as extilizações do nosso projeto.
