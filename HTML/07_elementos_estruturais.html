A Tag <div>

Estou dentro de uma <div>

A tag <div>, é muito util para conseguir organizar as sessões do nosso site, e também auxilia na hora de estilizar nosso site com CSS. Serve muito para segurar os elementos, como se fosse uma caixa.

Tag <span>

Assim como a tag <div>, a <span> também é invisivel, e também ajuda na hora de poder estilizar o elemento que estiver dentro dessa tag.

Estou dentro de uma <span>.
