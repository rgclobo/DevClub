Listas Ordenas e Não Ordenadas

Listas Ordenas

São Listas que são ordenadas por números ou letras, 1, 2, 3 ...a, b, c ...

5ª Série B

  1. Ana
  2. Bruno
  3. Carlos
  4. Daniela
  5. Flavio

Para começar uma lista ordenada usa a Tag <ol> e dentro dela é colocado as Tag's <li>, item por item, um de cada vez, assim como no exemplo acima.

Listas Não Ordenadas

Assim como as ordenadas, são listas, mas sem ordem definidas.

Lista de Supermercado

  • Arroz
  • Feijão
  • Carne
  • Iogurte
  • Café

Para começar uma lista não ordenada usa a Tag <ul>, e de igual forma dentro dela é colocado as Tag's <li>, item por item, um de cada vez, assim como no exemplo acima.
