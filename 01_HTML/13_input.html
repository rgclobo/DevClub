Input

A tag <input> é a responsável por criar as caixas que recebem informações nos sites.

Como a caixa que pesquisamos no google, as caixas de login e senha e qualquer lugar onde for necessário inserir informações.

É por essa Tag, que vai ser colocada as informações que são enviadas para um sistema, site e etc.

Existem várias propriedades para a tag:

  • type="" - É o tipo do input. Existem muitos tipos de type. Pode ser do tipo text, number, password, email, button, checkbox e etc ...
  • autocomplete - Completa os campos automaticamente, tendo variações dentro como: nome, email, endereço e etc ...
  • disabled - serve para desabilitar o input

    • - Input abaixo com a propriedade disable:



  • value="" - Vai inserir um valor no input que já virá definido
  • readonly - Vai fazer com que o input seja apenas para leitura, não permite que seja digitado nada. Mas não é como a aparencia do desabilitado.

    • - Input Abaixo com a propriedade value="" e readonly:



  • placeholder - É a propriedade que deixa a informação para que serve o campo que vai ser criado. Pode escrever o que desejar nele.

    • - Input abaixo com a propriedade placeholder:



  • autofocus - Essa propriedade, vai fazer com que o esse input já venha com um foco automatico ao abrir a página.

Exemplo abaixo de um Input:


    <input 
        autocomplete="name" 
        placeholder="Digite seu nome" 
        autofocus
    >


Algo muito importante que acompanha o <input>, é o <label>.

O label é a etiqueta do input, e através dele, conseguimos fazer a vinculação de ambos através do atributo for="" no label e informando esse atributo no id="" no input.

Exemplo abaixo:


    <label 
        for="rua">Rua:
    </label>

    <input 
        id="rua" 
        type="text"
        autocomplete="street-address" 
        placeholder="Digite sua rua"
    >

Alguns tipos de <input type="">

  • type="text" - É o input que recebe informações em geral.



  • type="color" - Vai permitir escolher uma cor em uma caixa em cores hexidecimais.



  • type="range" - Abre uma barra de range, aonde é possivel escolher o valor, ela permite alguns atributos como: min="0" (Valor minimo) max="10" (Valor máximo) value="0" (Valor Inicial) e entre outros.


  • type="password" - É o input para inserir senhas. Vai deixar os caracteres ocultos por se tratar de uma senha.



  • type="email" - É o input que vai receber os e-mails. Se ao validar o mesmo, não for um e-mail no formato seuemail@seuemail.com, vai negar o envio.



  • type="checkbox" - É o input de marcação de check. Veja exemplo:


  • type="file" - É o input que permite fazer o envio (upload) de arquivos. Ele permite o atributo 'multiple' que permite fazer o envio de multiplos arquivos. Permite o atributo 'accept', que vai setar o formatos dos arquivos aceitos.


  • type="number" - É o input que só permite números.



  • type="date" - É o input que vai permitir datas



  • type="radio" - É o input que permite selecionar opções.
