Button

A tag <button> é responsavel por duas coisas dentro de um formulário.

Limpar os dados desse formulário através da propriedade type="reset", que vai literalmente apagar tudo que tenha sido preenchido nesse formulário.

Enviar os dados que foram preenchidos no formulário, através da propriedade type="submit". Ao clicar, vai enviar os dados preenchidos para onde foi configurado na tag <form> .

Exemplo:


        <form">
        <button type="reset">Limpar</button>
        <button type="submit">Enviar</button>
        </form>

Existem mais outra propriedade para a tag <button> .

A propriedade type="button" que serve apenas como um botão mesmo. Usado mais de forma conjunta com JS. Podendo ser programado para chamar outras funções, se assim o mesmo for programado.

Um Exemplo é usar o type="button", com função "disabled, fazendo com que o botão só fique ativo após o usuário cumprir alguma circustancia programada pelo desenvolvedor.
